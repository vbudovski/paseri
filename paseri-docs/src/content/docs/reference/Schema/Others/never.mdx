This schema rejects all values, and corresponds to the `never` TypeScript type. It can be useful in object schemas to
forbid a specific key, ensuring that parsing fails if the key is present in the input.

import { Playground } from "../../../../../components/Playground.tsx";

## Example

```typescript
import * as p from '@vbudovski/paseri';

const schema = p.never();
const data = 'foo';

const result = schema.safeParse(data);
if (!result.ok) {
    // result.issue flags value as invalid.
}
```
