## Example

```typescript
import * as p from '@vbudovski/paseri';

const schema = p.duration();
const data = Temporal.Duration.from({ hours: 1 });

const result = schema.safeParse(data);
if (result.ok) {
    // result.value typed as `Temporal.Duration`.
}
```
