## Example

```typescript
import * as p from '@vbudovski/paseri';

const schema = p.plainDate();
const data = Temporal.PlainDate.from('2020-01-01');

const result = schema.safeParse(data);
if (result.ok) {
    // result.value typed as `Temporal.PlainDate`.
}
```


