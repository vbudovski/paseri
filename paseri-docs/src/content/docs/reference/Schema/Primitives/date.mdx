## Example

```typescript
import * as p from '@vbudovski/paseri';

const schema = p.date();
const data = new Date(2020, 0, 1);

const result = schema.safeParse(data);
if (result.ok) {
    // result.value typed as `Date`.
}
```

## Validators

### `minDate`

Date on or after `value`.

```typescript
p.date(p.minDate(new Date(2020, 0, 1)));
```

### `maxDate`

Date on or before `value`.

```typescript
p.date(p.maxDate(new Date(2020, 0, 1)));
```
