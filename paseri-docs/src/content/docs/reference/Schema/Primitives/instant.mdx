## Example

```typescript
import * as p from '@vbudovski/paseri';

const schema = p.instant();
const data = Temporal.Instant.from('2020-01-01T00:00:00Z');

const result = schema.safeParse(data);
if (result.ok) {
    // result.value typed as `Temporal.Instant`.
}
```

## Validators

### `min`

```typescript
p.instant().min(Temporal.Instant.from('2020-01-01T00:00:00Z'));
```

### `max`

Instant on or before `value`.

```typescript
p.instant().max(Temporal.Instant.from('2020-01-01T00:00:00Z'));
```
