## Example

```typescript
import * as p from '@vbudovski/paseri';

const schema = p.bigint();
const data = 456n;

const result = schema.safeParse(data);
if (result.ok) {
    // result.value typed as `bigint`.
}
```

## Validators

### `gte`

Greater than or equal to `value`.

```typescript
p.bigint().gte(5n);
```

### `gt`

Greater than `value`.

```typescript
p.bigint().gt(5n);
```

### `lte`

Less than or equal to `value`.

```typescript
p.bigint().lte(5n);
```

### `lt`

Less than `value`.

```typescript
p.bigint().lt(5n);
```
