## Example

```typescript
import * as p from '@vbudovski/paseri';

const schema = p.set(p.number())();
const data = new Set([1, 2, 3]);

const result = schema.safeParse(data);
if (result.ok) {
    // result.value typed as `Set<number>`.
}
```

## Validators

### `minSize`

Consists of at least `size` elements.

```typescript
p.set(p.number())(p.minSize(3));
```

### `maxSize`

Consists of at most `size` elements.

```typescript
p.set(p.number())(p.maxSize(3));
```

### Exact size

Consists of exactly `size` elements.

```typescript
p.set(p.number())(
    p.minSize(3),
    p.maxSize(3),
);
```
