## Example

```typescript
import * as p from '@vbudovski/paseri';

const schema = p.array(p.number())();
const data = [1, 2, 3];

const result = schema.safeParse(data);
if (result.ok) {
    // result.value typed as `number[]`.
}
```

## Validators

### `minLength`

Consists of at least `length` elements.

```typescript
p.array(p.number())(p.minLength(3));
```

### `maxLength`

Consists of at most `length` elements.

```typescript
p.array(p.number())(p.maxLength(3));
```

### Exact length

Consists of exactly `length` elements.

```typescript
p.array(p.number())(
    p.minLength(3),
    p.maxLength(3),
);
```
