## Example

```typescript
import * as p from '@vbudovski/paseri';

const schema = p.map(
    p.number(),
    p.string(),
)();
const data = new Map([
    [1, 'foo'],
    [2, 'bar'],
    [3, 'baz'],
]);

const result = schema.safeParse(data);
if (result.ok) {
    // result.value typed as `Map<number, string>`.
}
```

## Validators

### `minSize`

Consists of at least `size` elements.

```typescript
p.map(
    p.number(),
    p.string(),
)(p.minSize(3));
```

### `maxSize`

Consists of at most `size` elements.

```typescript
p.map(
    p.number(),
    p.string(),
)(p.maxSize(3));
```

### Exact size

Consists of exactly `size` elements.

```typescript
p.map(p.number(), p.string())(
    p.minSize(3),
    p.maxSize(3),
);
```
